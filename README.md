## 📖 项目简介
本项目是一个基于 AI 协作开发的个人效率工具箱。通过 **GitHub 作为方法**，记录我从零构建一个全栈/工具类应用的过程。这不仅是一个代码仓库，更是我与 AI 协同进化的技术实验场。

> **核心理念**：模块化开发、AI 驱动逻辑、GitHub Actions 自动化。

---

## 🚀 路线图与功能模块 (Roadmap)

本项目采用分阶段迭代模式，每一格热力图都代表一个功能点的演进：

### 第一阶段：基础设施 (Core Architecture)
- [ ] **项目脚手架搭建**：选择 Vite/React 或 Python/Typer 搭建基础架构。
- [ ] **命令/路由分发系统**：实现插件化加载机制。
- [ ] **GitHub Actions 配置**：实现提交代码自动进行 Lint 检查。

### 第二阶段：实用工具集 (Utility Modules)
- [ ] **Module A: 智能汇率/单位转换**
  - 调用第三方 API，支持实时汇率。
  - AI 增强：支持自然语言输入（如“100美金等于多少人民币”）。
- [ ] **Module B: Markdown 实时预览引擎**
  - 支持渲染 CommonMark 与 GitHub Flavored Markdown。
- [ ] **Module C: 任务管理 (Todo SQLite)**
  - 本地轻量化存储，支持任务持久化。
- [ ] **Module D: 自动化脚本库**
  - 自动桌面整理、文件批量重命名等。

### 第三阶段：AI 深度集成 (AI Integration)
- [ ] **AI Shell 助手**：在工具箱内直接调用 LLM 解析复杂指令。
- [ ] **智能日志分析**：自动汇总每日 Commit 记录并生成进度周报。
